# vscode-japanese-text

日本語を入力しやすい設定を提供するリポジトリ

## 推奨拡張

- テキスト校正くん：テキストの誤記チェック
- CharacterCount：文字数カウント

## 設定

- Vim拡張の無効化
- Word Spacesの設定に句読点追加

## 使い方・TIPS

- Alt(Option) + → or ← で句読点まで移動
